
## 1.Wykorzystane technologie:


**Technologie Backend:**
Restowe API napisane w Springu z wykorzystaniem następujących technologii:
- Spring Boot
- Hibernate
- REST API
- JDBC
- Maven
- Lombok
- Docker (Windows 10 Docker Desktop)
